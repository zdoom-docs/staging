# EPlayerGender

<!-- api-declaration -->
enum EPlayerGender;

<!-- api-definition -->
The available player genders. This is represented as `uint8`, so there
are 251 unused genders.

<!-- api-variants -->
#-
{GENDER_FEMALE}

Feminine human gender.
-#

#-
{GENDER_MALE}

Male human gender.
-#

#-
{GENDER_NEUTRAL}

Non-binary human gender.
-#

#-
{GENDER_OTHER}

Other unspecified human gender or non-human gender.
-#
