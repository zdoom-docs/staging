FindLumpNamespace

Show declaration 
enum Wads.FindLumpNamespace;

An enumeration used for specifying where to fuzzy search files.

Variants

Wads.GLOBALNAMESPACE

Searches only in the global namespace.

Wads.ANYNAMESPACE

Searches in any namespace.
